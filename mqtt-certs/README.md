# MQTT TLS certificates

Place the broker CA and server cert/key here.

Expected files (generated by scripts):
- `ca.crt`
- `ca.key`
- `server.crt`
- `server.key`

Generate with:
- PowerShell: `pwsh ./scripts/gen-mqtt-certs.ps1 -AltNames "mqtt-broker,localhost,127.0.0.1,marzocchi-tech.ewe-mulley.ts.net"`
- Bash: `./scripts/gen-mqtt-certs.sh "mqtt-broker,localhost,127.0.0.1,marzocchi-tech.ewe-mulley.ts.net"`

Include your broker MagicDNS name (and optionally the Tailscale IP) in the SAN list so HAOS can validate the broker certificate.
